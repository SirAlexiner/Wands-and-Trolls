Game Instructions

Main Menu

The main menu contains the following buttons:

  • New Game: Starts a new game.
  • Load Adventure: Loads a previously created story.
  • Create Adventure: Allows you to create a new story.
  • Settings: Adjust game settings such as volume, language, and AI-generated images.
  • Help: Displays these instructions.

Settings

In the settings menu, you can:

  • Adjust the music and sound effect volume.
  • Mute and unmute the music and sound effects.
  • Update the language (English, Spanish, German, or Norwegian).
  • Enable AI-generated images for the game's background.

New Game

When starting a new game, you can:

  • Choose an avatar image.
  • Enter your name.
  • Set your starting gold and health.
  • Define your goals for the game.

Gameplay

During gameplay:

  • Use buttons to navigate through the story.
  • A box displays the current room (passage) description.
  • Use a checkbox in the top left to view your inventory.
  • Your health, gold, and score are displayed at all times.

Game Over

If you fail, a defeated screen will be shown, allowing you to go back to the main menu. If you win, a victory page will be shown, and you can also go back to the main menu. During the game, you can restart the story at any point using the "restart" button.
