Game Instructions

Main Menu

The main menu contains the following buttons:

  • New Game: Starts a new game.
  • Load Adventure: Loads a story.
  • Settings: Adjust game settings such as volume, language, or AI-generated images.
  • Help: Displays these instructions.

Settings

In the settings menu, you can:

  • Adjust the music and sound effect volume.
  • Mute and unmute the music and sound effects.
  • Update the language (English, Spanish, German, or Norwegian).
  • Enable AI-generated images for the game's background.

New Game

When starting a new game, you can:

  • Choose an avatar image.
  • Enter your name.
  • Set your starting gold and health.
  • Define your goals for the game.

Gameplay

During gameplay:

  • Use buttons to navigate through the story.
  • A box displays the current room (passage) description.
  • Your health, gold, and score are displayed at all times.

Win Condition (Current version)

To win a game loaded into the current version of Wands and Trolls, make your eay through the paths until you find the last "passage". Upon which a page will be shown where you can go back to the main menu. During the game, you can restart the story at any point using the "restart" button, and, you can return to the main menu at any point using the "Main Menu" button, but this loses your progress.
